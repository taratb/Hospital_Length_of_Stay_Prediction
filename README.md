# Predviđanje trajanja hospitalizacije pacijenata

## Opis projekta
Projekat se bavi predviđanjem dužine hospitalizacije pacijenata
korišćenjem regresionih modela, kao i eksplorativnom analizom medicinskih i demografskih podataka.

## Dataset
Hospital Length of Stay Dataset (https://www.kaggle.com/datasets/aayushchou/hospital-length-of-stay-dataset-microsoft)

## Korišćeni modeli
- Random Forest
- XGBoost

## Evaluacija
- MSE
- RMSE
